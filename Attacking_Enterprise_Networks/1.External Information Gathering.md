# Lưu ý quan trọng
❗ Không nên dùng bừa
  Vì -sC có thể:
    tạo log
    trigger IDS/IPS
    bị coi là scan tấn công

🔐 Một số tính năng cần quyền cao:
  -O (OS detect)
  traceroute
    👉 nếu không chạy với root/admin → sẽ bị disable


🧪 Thực tế (best practice)
Thay vì -A, nhiều pentester thích control rõ hơn:
```
nmap -sV -sC -O <target>
```
  → tránh chạy những thứ không cần thiết



# Lệnh bạn chạy là gì
```
sudo nmap --open -p- -A -oA inlanefreight_ept_tcp_all_svc -iL scope
```
📌 Ý nghĩa nhanh:
Option	Ý nghĩa
--open	chỉ show port open
-p-	scan toàn bộ 65535 port
-A	aggressive (OS + version + script + traceroute)
-oA	lưu output (xml + grep + nmap)
-iL scope	đọc target từ file
