## Verb Tampering - Method

Cái này nó xoay quanh method GET PUT POST DELETE OPTION HEAD .... 

Ví dụ khi tôi muốn reset tài khoản admin bằng API /reset-passw/admin GET nhưng nó kiểm tra với authen hợp lệ, nhưng nếu có lỗi verb Tempering thì có thể dùng method ví dụ như HEAD sẽ có thể được chấp thuận request và hàm xử lý của API /reset-passw/admin sẽ chạy bình thường không có validate bên trong
